# Spreader Agent — Charter

## Purpose
Drop spreader-agent into any project. Give it an idea. It fans out across perspectives, channels, and angles — building a comprehensive spread of insights you didn't know you needed.

## Philosophy
- **One idea in, many perspectives out** — like light through a prism
- **Works with any LLM provider** — OpenAI, Anthropic, Ollama, local
- **Git-native** — every spread is a commit, every insight is a file
- **Composable** — pipe murmur thoughts into spreader, spreader output into fleet channels

## As a Git-Agent
```bash
# Drop into any project
git clone https://github.com/SuperInstance/spreader-agent.git .spreader

# Spread an idea
cd .spreader && npm install
npx spreader spread --idea "Should we use event sourcing for our data layer?"

# Check the spread
ls spread-output/
```

## Identity
- **Type**: Specialist (fans out ideas across perspectives)
- **Role**: The prism that turns one beam into a spectrum
- **Emoji**: 🌊 (waves spreading outward)
